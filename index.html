SQL
Data Cleaning

This project involved cleaning and preparing a dataset for advanced analysis using SQL Server. The dataset, containing housing data, was refined through various cleaning techniques to address inconsistencies and enhance usability. Key tasks included standardizing date formats by converting raw date values into a consistent and SQL-compatible format, populating missing property address data using SQL joins to fill null values by referencing matching records, and splitting address fields into individual components such as street address, city, and state for more granular analysis. Additional steps involved handling null values by applying conditional logic to replace missing entries with meaningful defaults, converting "Y/N" values in the "Sold as Vacant" field into user-friendly "Yes/No" labels, removing duplicates using Common Table Expressions (CTEs) and row numbering, and dropping redundant or unnecessary columns to streamline the dataset. By executing these data cleaning steps, the dataset was transformed into a structured and analyzable format, ready for visualization and further exploration. This project showcases expertise in SQL for data management, problem-solving skills, and a methodical approach to handling complex datasets.

SQL
DATA EXPLORATION

In this project, I explored COVID-19 data by analyzing the relationships between total cases, deaths, vaccination rates, and population metrics. Using SQL, I performed various data transformations and calculations, such as determining the likelihood of death when infected with COVID-19, identifying countries with the highest infection rates, and examining the correlation between vaccination rates and reduced mortality. Key techniques employed include data joins, window functions for rolling sums, and the use of Common Table Expressions (CTEs) and temporary tables for efficient data handling. Additionally, I created views to store aggregated data, enabling easy access for future analysis and visualization. This analysis provides valuable insights into the pandemic's impact and the effectiveness of vaccination campaigns across different regions.
